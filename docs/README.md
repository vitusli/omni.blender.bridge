# Blender Bridge

Adds a Blender Bridge extension for Isaac Sim.

## Usage

1. Add the parent folder of this extension as an extension search path in Isaac Sim.
2. Enable **Blender Bridge** in `Window > Extensions`.
3. Configure the Blender executable path in the extension settings.
4. Use the Blender Bridge UI to send selected meshes to Blender and bring edited meshes back.
